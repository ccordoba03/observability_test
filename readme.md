
# EKS & Observability Engineering Challenge

Este proyecto implementa una arquitectura de referencia en **AWS EKS** priorizando:

- **Observabilidad como Código** (Grafana Alloy + Amazon Managed Prometheus).
- **Escalado inteligente** con **Karpenter**.
- **Gestión segura de secretos** con **External Secrets Operator**.
- **Exposición de aplicación demo** vía **AWS Load Balancer Controller**.
